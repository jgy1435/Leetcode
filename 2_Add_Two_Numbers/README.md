Leetcode Problems
=====================
## Problem #2 Two_Numbers
### 사용한 알고리즘
* Singly Linked-List
### 문제해결방안
* 단방향 연결리스트를 사용하여 두 수를 더하는 문제이다. 어렵지 않은데 carry발생 시 적절한 대처가 중요하다.
* 코드가 다소 난잡하다. 조건이 딱히 없어서 예외상황이 생길 때마다 수정하느라 그런감이...
* time complexity를 증가시킨 코드가 하나 있는데, temp 노드를 만들어 현재 list의 끝을 찾는 코드를 매 iteration마다 수행했는데, 굳이 그럴필요가 없었다... 현재 list의 마지막을 가리키는 포인터를 만들기만 하면 됐는데 반성하자.
